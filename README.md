# Product CRUD API

A RESTful API for product management developed with Java, Spring Boot, PostgreSQL, and Maven, with endpoint testing performed using Insomnia.

This project was developed during a guided class led by the course instructor and served as a practical introduction to backend application development. Through its implementation, it was possible to apply essential concepts such as API structuring, HTTP request handling, database integration, and endpoint validation in a real development scenario.

## Overview

The application follows the CRUD model and allows basic product management operations through a REST API, including product creation, retrieval, update, and deletion.

## Technologies Used

- Java
- Spring Boot
- Maven
- PostgreSQL
- Insomnia
- IntelliJ IDEA

## Technical Concepts Applied

During the development of this project, the following backend concepts were applied and reinforced:

- RESTful API development
- CRUD operation implementation
- HTTP methods and request handling
- Integration with PostgreSQL
- Dependency management with Maven
- Endpoint testing with Insomnia
- Backend project structuring with Spring Boot

## Learning Experience

Although developed in a guided learning environment, this project provided valuable hands-on experience in backend development. It contributed to a better understanding of how a Java application interacts with a relational database, how REST endpoints are structured, and how testing tools support the validation of application behavior.

## Project Structure

The project follows a standard Java backend structure, including source code, Maven configuration, and the necessary files for execution and dependency management.

## Execution

To run the project, clone the repository, open it in IntelliJ IDEA, configure the database connection in the application settings, create the required database structure in PostgreSQL, and start the application. After that, the endpoints can be tested using Insomnia.

## Final Considerations

This project helped strengthen practical skills in Spring Boot application structuring, PostgreSQL integration, HTTP method usage, API testing workflows, and CRUD-based backend logic. Even though it was developed in a guided class, it represented an important learning experience in building backend applications with Java.
